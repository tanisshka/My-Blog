
# My Blog

A simple and dynamic blogging platform built with **Node.js, Express, EJS, and Axios**. Users can **create, edit, and delete blog posts** with a modern, colorful interface.

## Features
- Create, edit, and delete blog posts.
- Dynamic rendering with **EJS templates**.
- Fetch and update data using **Axios**.
- Responsive and colorful **CSS design** with hover effects.
- Full-stack integration between frontend and backend.

## Technologies Used
- **Node.js** and **Express** for server-side logic.
- **Axios** for API requests.
- **EJS** for dynamic templating.
- **CSS** for styling and responsive design.
- **In-memory API** for storing posts.

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/my-blog.git
````

2. Navigate to the project folder:

   ```bash
   cd my-blog
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the backend API server:

   ```bash
   node index.js
   ```
5. Start the frontend server:

   ```bash
   node server.js
   ```
6. Open your browser at `http://localhost:3000`.

## Usage

* Click **"New Post"** to add a post.
* Click **"Edit"** to modify a post.
* Click **"Delete"** to remove a post.
